🔍 获取指定日期的论文

在 GitHub 仓库手动触发 Workflow，填写日期即可：

  1. 打开 Actions 页面
  2. 选择 Daily arXiv Paper → 点击 Run workflow
  3. 在 date 输入框填入日期，格式 YYYY-MM-DD（如 2026-05-22）
  4. 点击 Run workflow，等待完成
